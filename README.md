# ♻️ SmartSweep

SmartSweep is a MERN stack web application for smart waste management. It helps optimize garbage collection using intelligent routing, bin tracking, and complaint management.

---

## 🚀 Features

* 🗑️ Manage waste bins
* 🛣️ Optimized collection routes
* 📊 Complaint tracking
* 🤖 ML-based intelligence API

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Other: Mongoose, Axios

---

## ⚙️ Setup

### Backend

```
cd backend
npm install
npm start
```

### Frontend

```
cd frontend
npm install
npm start
```
